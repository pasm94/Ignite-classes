## CHAPTER I
#### Fundamentos do ReactJS
* Criando estrutura do projeto
  * yarn init -Y
  * yarn add react
  * yarn add react-dom

* Configurando Babel
  * yarn add @babel/core @babel/cli @babel/preset-env -D
  * yarn babel src/index.jsx --out-file dist/bundle.js
  * yarn add @babel/preset-react -D
